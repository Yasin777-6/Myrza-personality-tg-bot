# Telegram Bot "Мырза" (Myrza)

Это Telegram-бот на Python, который имитирует твоего саркастичного одноклассника **Мырзу** (учился в школе Ага Хана в Оше, с 6 по 9 класс). Он общается на русском языке, обожает CS:GO, Genshin Impact, крипту и архитектуру, и безбожно стебет участников группы.

## 🧠 Личность Мырзы
- **Сарказм и стеб**: Бот отвечает с цинизмом, высмеивает душность, крипту, киберспорт или глупые вопросы.
- **Индивидуальный подход (Кто есть кто)**: Бот распознает участников группы по их Telegram username или ID:
  - **Артур** — его обожаемый авторитет. Бот беспрекословно слушается Артура, соглашается с ним и подлизывается («Да, Артур», «Конечно, Артур»).
  - **Актан** — его любимая жертва для шуток про «Нуни» (мем с пальцами из Наруто).
  - **Ясин** — старый друг, с которым они вместе ходили из школы Aga Khan.
  - **Мансур и Бека** — друзья-одноклассники, которых он стебет.
  - **Все остальные** — обычные участники чата, над которыми он подшучивает с высокой колокольни.
- **Школьные мемы**: Помнит, как проиграл спор на **100 щелбанов** по лбу.

## 🛠️ Предварительная настройка

### 1. Создание Telegram Бота
1. Найдите бота [@BotFather](https://t.me/BotFather) в Telegram.
2. Отправьте команду `/newbot` и следуйте инструкциям, чтобы создать бота и получить **API Token**.
3. **Критически важно (Отключение приватности)**:
   По умолчанию боты в группах видят только сообщения, начинающиеся с `/` или прямые упоминания. Чтобы Мырза читал все сообщения и шутил случайно:
   - Отправьте `/mybots` боту `@BotFather`.
   - Выберите вашего созданного бота.
   - Перейдите в **Bot Settings** -> **Group Privacy**.
   - Нажмите **Turn off** (должно появиться сообщение `Privacy mode is disabled`).
4. Добавьте бота в вашу группу.

### 2. Получение DeepSeek API Key
1. Перейдите на [DeepSeek Platform](https://platform.deepseek.com/) и зарегистрируйте аккаунт.
2. Перейдите в раздел **API Keys** и создайте новый ключ.

## 💻 Локальный запуск (для тестирования)

1. Установите Python 3.10+ на компьютер.
2. Склонируйте или скачайте этот проект.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Создайте файл `.env` на основе `.env.example`:
   ```bash
   copy .env.example .env
   ```
5. Откройте `.env` и заполните:
   - `TELEGRAM_BOT_TOKEN`
   - `DEEPSEEK_API_KEY`
   - Юзернеймы друзей в Telegram (без символа `@`, строчными буквами), например: `USER_ARTUR=artur_tg,12345678` (можно вводить как username, так и числовой Telegram ID).
6. Запустите бота:
   ```bash
   python bot.py
   ```

## 🚀 Бесплатный деплой (24/7)

Вы можете запустить этого бота абсолютно бесплатно в облаке без необходимости держать свой компьютер включенным. Вот лучшие бесплатные платформы:

### Вариант 1: Koyeb (Рекомендуется)
Koyeb предлагает отличный бесплатный тариф, который поддерживает Docker-контейнеры и Git-деплой.
1. Создайте аккаунт на [Koyeb](https://www.koyeb.com/).
2. Создайте новый репозиторий на GitHub и загрузите туда файлы проекта (`bot.py`, `requirements.txt`, `Dockerfile`).
3. На Koyeb нажмите **Create App**.
4. Выберите **GitHub** в качестве источника и укажите ваш репозиторий.
5. В настройках деплоя выберите тип **Worker** (или Web Service, если Koyeb требует этого; если Web Service, укажите любой порт, например 8000, бот будет работать через Polling в фоне).
6. В разделе **Environment Variables** добавьте переменные из файла `.env`:
   - `TELEGRAM_BOT_TOKEN`
   - `DEEPSEEK_API_KEY`
   - `REPLY_PROBABILITY` (например, `0.15`)
   - `USER_ARTUR`, `USER_AKTAN`, `USER_YASIN`, `USER_MANSUR`, `USER_BEKA`
7. Нажмите **Deploy**. Бот запустится и будет работать постоянно в фоне.

### Вариант 2: Hugging Face Spaces (Бесплатный Docker)
Hugging Face предоставляет бесплатные Spaces, которые могут круглосуточно запускать Docker-контейнеры.
1. Зарегистрируйтесь на [Hugging Face](https://huggingface.co/).
2. Нажмите **New Space**.
3. Укажите любое имя, выберите SDK: **Docker** (шаблон Blank).
4. Сделайте Space **Public** или **Private** (лучше Public, чтобы он не уходил в спящий режим слишком быстро, или настройте активность).
5. Перейдите в **Settings** вашего Space.
6. Найдите раздел **Variables and Secrets** и нажмите **New Secret** для добавления переменных окружения:
   - Добавьте `TELEGRAM_BOT_TOKEN`, `DEEPSEEK_API_KEY`, `USER_ARTUR` и т.д.
7. Перейдите во вкладку **Files and versions**, нажмите **Add file** -> **Upload files** и загрузите `bot.py`, `requirements.txt`, `Dockerfile`.
8. После загрузки файлов Hugging Face автоматически начнет сборку Docker-образа и запустит вашего бота.

### Вариант 3: Render (Web Service с пингом)
Render предоставляет бесплатные Web Services, но они засыпают через 15 минут простоя. Для бота на Polling это неудобно, но можно использовать webhook, либо сделать простой трюк с пингом:
1. Создайте репозиторий на GitHub с файлами бота.
2. Зарегистрируйтесь на [Render](https://render.com/).
3. Нажмите **New** -> **Web Service**.
4. Подключите ваш GitHub репозиторий.
5. Установите Build Command: `pip install -r requirements.txt` и Start Command: `python bot.py`.
6. Добавьте ваши переменные окружения в разделе **Env**.
7. Чтобы бот не засыпал, вы можете использовать бесплатный внешний сервис (например, [UptimeRobot](https://uptimerobot.com/)), который будет пинговать URL вашего Render-приложения каждые 10 минут.
